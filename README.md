## Do you wish there where more mob head trophies in Minecraft? 

What if every mob could drop it's own unique head? Would'nt it be cool if these trophies had accurate models? This Minecraft Bedrock addon aims to fill the gap left by Mojang when it comes to these very cool decorational items.

Video Showcase: https://youtu.be/1CfagoLt-zg?si=YoXGs_qf8rYKP2EU

## Installation
### Releases
Simply download the latest mcaddon file from the releases tab, or visit [MCPEDL](https://mcpedl.com/vanilla-mob-heads/)!
### Building from Source
Download the source and extract it, then run createHeads.py (you're gonna need python). The script will then generate the RP and BP folders for you to import into Minecraft. Make sure to delete these two folders before running the script again.
